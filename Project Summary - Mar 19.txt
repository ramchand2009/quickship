This project is a Django 6 operations platform for Mathukai order management. It manages orders, inventory, packing, shipping labels, WhatsApp notifications, webhooks, and admin diagnostics.
Main Purpose
It helps an operations team handle ecommerce orders from intake through packing, shipping, delivery, completion, cancellation, and customer notification.
Core Features
Order management
Syncs/imports orders from Shiprocket and WooCommerce
Tracks local order lifecycle: new, accepted, packed, shipped, delivery issue, out for delivery, delivered, completed, cancelled
Supports manual order/customer/address edits before shipment
Includes bulk status updates, saved views, undo, CSV export, and order detail pages

Packing and print workflows
Packing queue
Packing lists
Barcode/scan-based packing validation
4x6 shipping labels
Bulk labels and PDF generation
Print tracking/counts

Inventory / stock
Product catalog with SKU, barcode, category, image, stock quantity, reorder level
Stock deduction/restoration tied to order status transitions
Manual stock movements
Special stock issue register for free/sample/complimentary stock

WhatsApp notifications
Integrates with Whatomate / WhatsApp APIs
Sends order status updates, payment reminders, test messages/templates
Has queue processing with retries, failed queue handling, delivery logs, webhook updates, and template configuration

Webhooks
Whatomate webhook for WhatsApp delivery/incoming status
WooCommerce webhook for order updates
Webhook diagnostics and stale webhook warnings

Operations/admin tooling
Dashboard with queue, health, stock, and order status views
Role-based access: admin and read-only ops_viewer
Health endpoint: /healthz/
Metrics endpoint: /metrics/
Integration smoke checks
Audit CSV export
Backup/restore commands
Error digest and incident snapshot commands
Login lockout/rate limiting

Tech Stack
Backend: Django 6.0.2
Database: SQLite locally, PostgreSQL supported via env/database URL
Static files: WhiteNoise
PDF labels: ReportLab
Push notifications: pywebpush
Deployment support: Docker, docker-compose, Gunicorn, Hostinger/EasyPanel docs
Important Folders
core/ - main Django app: models, views, integrations, stock, queue, monitoring
templates/core/ - UI pages for orders, stock, labels, WhatsApp, dashboards
static/ - CSS, JS, PWA assets, admin theme assets
scripts/ - Windows scheduled task helpers for backup and WhatsApp worker
docker/ - container entrypoint
backups/, logs/ - runtime operational data
