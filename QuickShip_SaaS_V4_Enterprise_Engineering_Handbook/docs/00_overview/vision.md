# Vision

QuickShip is the operational control room for Mathukai order fulfilment. The product focuses on reducing manual order handling errors by bringing ecommerce orders, stock movement, packing validation, shipping labels, WhatsApp updates, and profit visibility into one Django dashboard.

The current system is built for a single operations team with two working modes:

- Desktop admin: broader management, configuration, exports, diagnostics, stock/product maintenance, WhatsApp settings, and utilities.
- Mobile operations: fast order acceptance, delivery detail correction, packing scan validation, shipping updates, label workflows, and queue visibility.

The main business objective is dependable fulfilment: accept orders only when delivery data and stock are workable, pack only after all expected products are scanned, ship with tracking and shipping cost captured, keep customers updated through WhatsApp, and keep profit/stock numbers visible.

Future SaaS/enterprise ideas exist in this handbook, but the live code is currently a single-business operations dashboard, not a tenant-isolated SaaS platform.
